what is the difference between accessing data from Arrays and objects?

The difference between accessing data from Arrays and objects is that arrays store items with and index, that is why in order to retrieve a piece of information from an array us must use and index. However when you get data from an object they come in a key value pair, that means in order to get a piece of data, you refer to it using a key.

what is a function and why are they useful

We can think of functions as a set procedure for excecuting tasks. An example of this could be making a cheeseburger, there would be set steps for making a cheeseburger, such as first bun, then patty, then cheese, then sauce, then other bun, and task complete. functions are very useful because they are reusable, instead of having to recode the same procedure over and over, we can simply call upon the function again and it will execute, (DRY = Don't Repeat Yourself).
